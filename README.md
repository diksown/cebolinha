# cebolinha

[![](https://travis-ci.org/theuves/cebolinha.svg?branch=master)]()

> Fale como o Cebolinha.

## Instalação

```
npm install cebolinha
```

## Exemplos

```js
const cebolinha = require('cebolinha');

cebolinha("Prato"); // "Plato"
cebolinha("Carlos"); // "Caulos"
cebolinha("Comer"); // "Comer"
```
